# Assessly Screenshots Guide

This directory contains screenshots for the README.md documentation. Place your screenshot files here following this naming convention:

## Required Screenshots

1. **dashboard.png** — Admin dashboard showing supply chain risk score, compliance metrics, and assessment status overview
2. **vendors-table.png** — Vendor management table with search, filtering, access codes, and action buttons
3. **send-invite.png** — Modal dialog for sending secure vendor invitations with email/SMS split delivery
4. **assessment-workspace.png** — Internal assessment page showing NIS2 questionnaire with embedded document audit sidebar
5. **document-audit.png** — Document upload zone with AI audit results and compliance analysis
6. **vendor-portal-login.png** — External vendor portal login page for accessing isolated assessment sessions
7. **external-assessment.png** — Vendor-facing assessment interface showing questionnaire and evidence upload
8. **audit-trail.png** — Audit trail page showing timestamped security events with forensic metadata

## Recommended Format

- **Resolution**: 1280×720px or higher (16:9 aspect ratio)
- **Format**: PNG or JPG
- **File size**: < 500KB (compress if needed)
- **Content**: Capture full feature demonstrations in light mode or both light/dark modes

## How Screenshots Are Referenced

All screenshots are linked in the main README.md under the "Visual Overview" section. When you add screenshot files to this folder, the README will automatically display them.

Example reference:
```markdown
![Screenshot Title](docs/screenshots/filename.png)
```
